# مراحل ساخت فایل ترجمه

1. به مسیر زیر بروید:  
   `The Coffin of Andy and Leyley\www\languages\tool`

2. فایل **`Translator.exe`** را اجرا کنید.

3. گزینه **`Build`** را انتخاب کنید.

4. فایل **`dialogue.txt`** را انتخاب نمایید.

5. مسیر زیر را به عنوان مقصد تنظیم کنید:  
   `The Coffin of Andy and Leyley\www\languages`

6. دکمه **`Build`** را بزنید.
