# مراحل ساخت فایل ترجمه

اول به مسیر زیر بروید:  
   `The Coffin of Andy and Leyley\www\languages\tool`

دوم فایل **`Translator.exe`** را اجرا کنید.

سوم گزینه **`Build`** را انتخاب کنید.

چهارم فایل **`dialogue.txt`** را انتخاب نمایید.

پنجم مسیر زیر را به عنوان مقصد تنظیم کنید:  
   `The Coffin of Andy and Leyley\www\languages`

ششم دکمه **`Build`** را بزنید.
